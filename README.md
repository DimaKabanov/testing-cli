CLI testing
=========

## Commands
- `make install` - Install all dependencies
- `make start` - Run code
- `make build` - Transpile ES6 => ES5
- `make test` - Run all tests
- `make test-watch` - Run all tests with `watch` mode
- `make lint` - Run ESlint with
- `make publish` - Publish the package to NPM
